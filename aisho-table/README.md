# 相性表ツール 説明書

このファイルは、デッキ同士の相性表管理ツールです。

## 1. 画面の見方

### 保存済み表
- **名前を付けて保存**: 現在の表を新しい名前で保存します。
- **上書き保存**: 現在選択中の保存表に上書きします。
- **読み込み**: プルダウンで選んだ保存表を読み込みます。
- **削除**: プルダウンで選んだ保存表を削除します。
- **CSV保存**: 現在の表をCSVファイルとして出力します。
- **CSV読込**: CSVファイルを読み込んで表へ反映します。

### デッキ追加
- クラスを選んで **デッキ追加** を押すと、表に新しいデッキが追加されます。

### 相性表
- 行: 自分が使うデッキ
- 列: 対面デッキ
- セルをクリックすると、勝率を `- / 0% / 10% / ... / 100%` から選べます。
- 片側を入力すると、反対側は自動で `100 - 入力値` になります。
  - 例: A→B を 60% にすると、B→A は 40% になります。
- 対角線は同一デッキ対決なので `—` 表示です。

### 表右下のボタン
- **表リセット**: デッキ構成は残したまま、相性データだけ消します。
- **初期状態に戻す**: デッキ一覧を初期7クラスに戻し、相性表も初期化します。

### BO1先出し推奨(テスト)
- 自分の2デッキと、相手の公開2クラスを選択して **先出し計算** を押します。
- 相手候補に対する平均勝率から、どちらを先に出すべきかを表示します。

## 2. 保存の仕様

### IndexedDB保存
ブラウザ内の IndexedDB に保存されます。
- ブラウザを閉じても通常は残ります。
- ただし、ブラウザのデータ削除やサイトデータ消去を行うと消える場合があります。

### 自動保存
次の操作は、**現在選択中の保存表がある場合のみ自動で上書き保存** されます。
- デッキ追加
- デッキ削除
- デッキ順移動
- デッキ名変更
- セル値変更
- 表リセット

### 自動保存されない扱い
次の操作では、現在の保存名を外して「未保存の表」として扱います。
- CSV読込
- 初期状態に戻す

この状態で保存したい場合は、**名前を付けて保存** を使ってください。

## 3. 色の見方
- 0〜20%: 虹色アニメーション
- 30〜40%: 寒色寄り
- 50%: 緑
- 60%以上: 赤系
- 100%: 濃い赤

凡例も相性表と同じ色ルールになるようにしてあります。

## 6. 注意点
- 同じ名前で「名前を付けて保存」すると、IndexedDBの仕様上は実質上書きになります。
- SVGアイコンファイル (`icon_e.svg` など) はHTMLと同じ場所に置く想定です。
